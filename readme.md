## Local Setting Guide

### Prerequisites
install the following libraries:

* python -m venv venv # 파이썬 가상환경 만들기

* source ./venv/bin/activate # 가상환경 활성화

* pip install pytest


### S반점 FastAPI 실행 방법: 

커맨드라인에서 다음 명령어 실행 

uvicorn src.employee.jumwon:app --reload

후 브라우저에서 다음 URL로 접근.

(직접 호출) http://127.0.0.1:8000 

(Swagger 보기) http://127.0.0.1:8000/docs
